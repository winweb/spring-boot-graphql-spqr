## test case
### case 1 when you want find all cars.
```
query {
  cars {
    id
    name
  }
}
```

### case 2 when you want save new car.
```
mutation {
  saveCar(car: { name: "New Car" }) {
    id
    name
  }
}
```

### case 3 when you want update some car.
```
mutation {
  saveCar(car: { id: 9, name: "New Car" }) {
    id
    name
  }
}
```

### case 4 when you want inquiry some car.
```
query {
  car(id: 9){
    id
    name
    isCool
    giphyUrl
  }
}
```
